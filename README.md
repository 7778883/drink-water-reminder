# 喝水小助手

Windows 桌面端定时喝水提醒工具。用「树苗成长 vs 枯萎」做视觉反馈，靠内生动力促进养成喝水习惯。

## 功能特性

### 核心循环
- **定时提醒**：默认每 60 分钟（可在设置里改 15-180 分钟）弹出提醒，主窗口进入响应态
- **3 种响应方式** + **主动喝水**：
  - 提醒态 3 快捷按钮：`喝一口 +50ml` / `喝半杯` / `喝一杯`
  - 常态 3 主动按钮：`主动喝一口` / `主动喝半杯` / `主动喝一杯`（不受提醒周期限制，想喝就喝）
  - `推迟 5 分` / `跳过` 两个响应选项
- **单次幂等**：一次提醒只能记账一次，防止重复累加
- **跨日重置**：跨日自动归档昨天数据到 `history.json`，今日重新开始

### 视觉反馈（内生动力机制）
- **28 张水彩手绘图**：9 个成长阶段 × 3 种枯萎状态 + 1 张果实图
- **stage 0-8**：进度从 0% 到 88% 每 11.1% 一档，均匀分布
  - stage 0（种子）→ stage 1（嫩芽）→ ... → stage 8（大树）
- **枯萎状态**：
  - S0 健康 → S1 微枯 → S2 严重枯萎
  - 连续跳过 3 次触发视觉重置（drunk_ml 保留但视觉回种子）
- **达成目标**：喝够每日目标，显示果实累累的果树 `stage9_s0.jpg`

### 系统集成
- **系统托盘**：主窗口关闭时最小化到托盘，右键菜单可退出
- **托盘 flash + 前置**：提醒到点时若主窗口在托盘，图标闪烁 30 秒，超时自动前置
- **开机自启动**：设置里勾选后自动写入 Windows 启动文件夹
- **首次启动 Toast**：欢迎信息 + 下次提醒时刻

### 数据持久化
- `config.json`：设置 + 当日状态
- `reminders.json`：每次提醒的完整生命周期记录
- `drink_entries.json`：每次喝水事件（时间、量、来源）
- `history.json`：跨日凝结的历史快照

## 技术栈

- **语言**：Python 3.10+
- **GUI 框架**：PySide6 (Qt 6 Python 绑定)
- **打包工具**：PyInstaller 6.x
- **架构**：单文件 `main.py` 约 1600 行 + 独立数据 JSON

## 怎么运行

### 方式 1：下载打包好的 exe（推荐给普通用户）

从 Release 里下载 `喝水小助手_v1.0.zip`，解压后双击 `喝水小助手.exe`。

### 方式 2：源码运行（需要 Python）

```powershell
pip install PySide6
python main.py
```

或者双击项目根目录的 `start.bat`（用 pythonw 后台启动，无控制台窗口）。

## 使用说明

1. **主窗口顶部**：显示「今日 X / Y ml」进度 + 状态文字 + 倒计时
2. **画布区**：显示当前状态对应的水彩图
3. **底部按钮区**：
   - 常态：3 个「主动喝一口/半杯/一杯」按钮
   - 提醒态：3 个「喝一口/半杯/一杯」按钮 + 推迟/跳过
4. **右上齿轮**：进入设置，可改提醒间隔、单杯 ml、每日目标、开机自启动
